# Roksi Shop API
DRF API for an online art shop (DEPLOYED VERSION).

## Features
* JWT Authentication
* Payment system LiqPay (Currently disabled)
* Email notifications
* Session-based cart

## Installation

#### Docker should be installed.

* Populate .env file according to .env.sample.

```commandline
docker-compose up --build
```
